• ajax

    ajax(option)，ajax异步请求。option是一个对象，拥有参数url, type, async, dataType, data, beforeSend, success, error。
    url：必需。字符串，ajax请求的地址。可以是url服务端请求地址，也可以是json、xml、html、txt文件地址
    type：可选。字符串，ajax请求的方式get/post。默认post
    async：可选。布尔值，是否异步，true异步/false同步。默认true
    cache：可选。布尔值，是否缓存，true开启缓存/false不缓存。默认true
    dataType：可选。字符串，请求返回的数据类型，text/json/xml，默认text。如果参数成jsonp，代表jsonp跨域请求
    data：可选。对象，请求是发送给服务端的数据，默认为空。
    beforeSend：可选。函数，请求发送之前执行。
    success：可选。函数，请求成功之后执行，函数参数一是服务端返回的数据，二是XMLHttpRequest对象。
    error：可选。函数，请求错误之后执行。函数参数一是错误状态码，二是XMLHttpRequest对象。
    示例：
    Gj.ajax({
    url: 'myplace.xml',
    type: "get",
    dataType: "xml",
    success: function(data,XMLHttp){
    }
    });

  • get

    get(url, data, fn)，ajax异步get请求。
    url：必需。字符串，ajax请求的地址。可以是url服务端请求地址，get方式下也可以是json、xml、html、txt文件地址
    data：可选。对象，请求是发送给服务端的数据，默认为空。
    fn：可选。函数，请求之后执行。
    示例：
    Gj.get('readme.txt',function(data){
    console.log(data);
    });
