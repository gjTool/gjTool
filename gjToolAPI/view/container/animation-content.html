• animate

    animate(styles, time, easing, callback)， CSS属性的自定义动画。
    styles：必需。json对象，规定产生动画效果的 CSS 样式和值。
    time: 可选。数字，毫秒单位，规定动画效果的持续时间，默认400。
    easing: 可选。字符串，linear, easeIn, easeOut, easeInOut规定动画效果的运动曲线，默认easeInOut。
    callback: 可选。回调函数，规定动画完成后的执行函数。
    示例：
    Gj(".container").animate({
    scrollTop: 0
    },1000,"easeIn");

  • show

  • fadeTo

    fadeTo(value, time, easing, callback)，设置元素的透明度以动画效果展示。
    value: 必需。数字，0~1，规定元素的透明度数值。
    time: 必需。数字，毫秒单位，规定动画效果的持续时间，默认400。
    easing: 可选。字符串，linear, easeIn, easeOut, easeInOut规定动画效果的运动曲线，默认easeInOut。
    callback: 可选。回调函数，规定动画完成后的执行函数。
    示例：
    Gj(".container").fadeTo(0.3);

  • stop

    stop(stopAll, goToEnd)，停止动画队列。
    stopAll: 可选。true，停止元素动画队列所有动画。为空的时候，停止元素动画队列当前动画，其它继续
    goToEnd: 可选。true，stopAll为true的时候才可设置。让元素动画队列当前动画完成，其它停止<
    示例：
    Gj(".container").stop(true);
