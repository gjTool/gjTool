• 文档加载完成

    Gj(function(){});Gj(document).ready(fucntion(){});
    两种方式都文档加载完成。在function内部写代码，就是代表文档加载完成后执行代码。

  • on

    on(eventType, selector, fn)，on事件绑定，给元素绑定事件。
    eventType：必需。字符串，事件名。
    selector: 可选。字符串，选择器名，代表事件监听的元素，有这个参数代表开启事件监听。
    fn: 必需。函数，在内部写代码表示事件执行要做什么，fn有可选参数，经过封装的event事件对象，兼容ie浏览器。
    示例：
    Gj("#div").on("click",function(e){});

  • off

    off(eventType)，off事件解绑，给元素解绑事件。
    eventType：必需。字符串，事件名。
    示例：
    Gj("#div").off("click");

  • hover

    hover(fn1, fn2)，hover鼠标划入划出事件。
    fn1：必需。函数，鼠标划入事件，mouseenter。
    fn2：可选。函数，鼠标划出事件，mouseleave。
    示例：
    Gj("#div").hover(function(){}, function(){});

  • 其他事件

    其他事件绑定。
    如："blur focus input focusin focusout load resize scroll unload click dblclick mousedown mouseup mousemove mouseover mouseout mouseenter mouseleave change select submit keydown keypress keyup error contextmenu"
    示例：
    Gj("#div").click(function(){});
