• 如何扩展插件方法

    extend扩展插件。Gj.fn.protoType或者Gj.protoType上添加方法。
    示例：
    Gj.protoType.alert = function(a){
    alert(a);
    }
    Gj.fn.protoType.alert = function(){
    alert(Gj(this).val());
    }
    Gj.extend({
    alert: function(){
    alert(a);
    } })
    Gj.fn.extend({
    alert : function(){
    alert(Gj(this).val());
    }

  • drag

    drag()，拖拽元素。nodrag(true)、nodrag()，元素关闭拖拽。nodrag(false)，元素开启拖拽。
    示例：
    Gj("#div").drag();//开启拖拽
    Gj("#div").nodrag();//关闭拖拽
    Gj("#div").nodrag(false);//开启拖拽
