    Gj.extend({

    }

  • now

    now()，获取当前时间戳，毫秒。 示例：
    Gj.now();

  • getTime

    getTime()，获取当前时间的年月日星期时分秒对象合集。
    示例：
    var obj = Gj.getTime();
    obj.y 年
    obj.m 月
    obj.d 日
    obj.h 时
    obj.mi 分
    obj.s 秒
    obj.w 星期

  • getDate

    getDate(w)，获取一个当前时间年月日星期。
    示例：
    var date = Gj.getDate();
    console.log(date);//2018年02月14日星期三 10:28
    var date = Gj.getDate("-");
    console.log(date);//2018-02-14 10:28
    var date = Gj.getDate();
    console.log(date);//2018/02/14 10:28

  • arrSort

    arrSort(arr, name, reverse)，数组按对象键名排序，升序。返回新数组
    arr：必需。数组，要排序的数组对象
    name：必需。数组对象的属性名，以对应的属性值来排序
    reverse：可选。布尔值，true代表降序排序
    示例：
    var arr = [{abc:123},{abc:345},{abc:111}];
    var newArr = Gj.arrSort(arr,"abc");//[{abc:111},{abc:123},{abc:345}]

  • cloneArr

    cloneArr(arr)，深度克隆数组。返回新数组
    arr：必需。数组，需要深度克隆的数组
    示例：
    var arr = [{abc:123},{abc:345},{abc:111}];
    var newArr = Gj.cloneArr(arr);//[{abc:123},{abc:345},{abc:111}]

  • cloneObj

    cloneObj(obj)，深度克隆对象。返回新对象
    obj：必需。对象，需要深度克隆的对象
    示例：
    var obj = {abc:111}
    var newObj = Gj.cloneObj(obj);//{abc:111}

  • extendArr

    extendArr(arr1, arr2, arr3)，差集合并数组，可以合并多个。返回新数组
    arr1：必需。数组，需要合并的数组
    arr2：必需。数组，需要合并的数组
    arr3：可选。数组，需要合并的数组
    示例：
    var arr1 = [{abc:123},{abc:345},{abc:111}];
    var arr2 = [{abc:123},{abc:666},{abc:111}];
    var newArr = Gj.extendArr(arr1, arr2);//[{abc:123},{abc:345},{abc:111},{abc:666}]

  • unique

    unique(arr)，数组去重。返回新数组
    arr：必需。数组，需要去重的数组
    示例：
    var arr = [1,1,2,3];
    var newArr = Gj.unique(arr);//[1,2,3];

  • objSort

  • browser

    browser 获取当前浏览器版本信息。browser是一个对象，可能包含的属性：opera，firefox，chrome，safari，webkit，ie
    示例：
    var browser = Gj.browser;
    console.log(browser.chrome);
    console.log(browser.webkit);

  • encrpt

    encrpt(str)，对字符串进行简单加密。
    示例：
    var newStr = Gj.encrpt("123");//%ez%fa%fl

  • decrpty

    decrpty(str)，对字符串进行简单加密。
    示例：
    var newStr = Gj.encrpt("123");//%ez%fa%fl
    var str = Gj.encrpt(newStr);//123

  • setCookie

    setCookie(name, value, hour)，存储cookie，经过encrpt加密存储到根目录。
    name：必需。字符串，需要存储的cookie名
    value：必需。字符串，需要存储的cookie值
    hour：必需。数字，单位小时。需要存储多久时间
    示例：
    Gj.setCookie("user", "123", 1);

  • getCookie

    getCookie(name)，按cookie名读取cookie值。
    name：必需。字符串，需要读取的cookie名
    示例：
    Gj.getCookie("user");

  • delCookie

    delCookie(name)，按cookie名删除cookie。
    name：必需。字符串，需要删除的cookie名
    示例：
    Gj.delCookie("user")

  • getVerify

    getVerify(length)，获取一定长度的随机数字字母。
    lenth: 必需。数字，获取随机数字字母的长度
    示例：
    var str = Gj.getVerify(4); //fDcR

  • easing

    easing，动画相关的运动曲线对象。可通过Gj.extend扩展。目前有属性：
    linear: 匀速
    easeIn: 加速
    easeOut: 减速
    easeInOut: 加速减速

  • 判断变量类型

    判断各种变量类型：都需要传入一个需要判断的参数，返回布尔值，true/false。
    Gj.isString(test): 判断是否是字符串
    Gj.isUndefined(test): 判断是否是undefined
    Gj.isNull(test): 判断是否是null
    Gj.isNumber(test): 判断是否是数字
    Gj.isBoolean(test): 判断是否是布尔值
    Gj.isDate(test): 判断是否是时间
    Gj.isObject(test): 判断是否是对象
    Gj.isArray(test): 判断是否是数组
    Gj.isFunction(test): 判断是否是函数
    Gj.isRegExp(test): 判断是否是正则
    Gj.isWindow(test): 判断是否是window对象
    Gj.isHTMLDocument(test): 判断是否是document对象
    Gj.isHTMLElement(test): 判断是否是dom元素
    Gj.isJSON(test): 判断是否是json字符串
